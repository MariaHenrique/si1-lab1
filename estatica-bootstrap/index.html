Por que cursar a disciplina SI 1?


O que significa SI 1?

SI 1 é a disciplina Sistemas de Informação 1, que de acordo com sua ementa deveria (e irá brevemente) ser chamada de Design de Software.


Objetivos:

Discutir técnicas de projeto de software, com foco particular em técnicas utilizadas para melhorar o reuso e a flexibilidade de software desenvolvido usando Orientação a Objeto. Também será discutido projeto arquitetural. O contexto de todas as aplicações será a web. Assim, a disciplina também acaba oferecendo o desenvolvimento de sistemas web.


O que você será capaz ao sair da disciplina?

  1. Descrever os objetivos do projeto de software e seu papel no desenvolvimento de um software;
  2. Definir o que são padrões de projeto e nomear um conjunto de padrões;
  3. Criticar a necessidade e/ou oportunidade da aplicação de padrões em um contexto;
  4. Descrever e criticar uma arquitetura de software;
  5. Aplicar boas práticas e padrões de projeto na transformação de uma especificação em um projeto flexível e reutilizável;
  6. Produzir um software complexo utilizando boas práticas e padrões de projeto;
  7. Produzir um software web que utilize tecnologias atualmente populares no mercado;
  8. Conectar o software sendo implementado às necessidades do cliente que o demanda;
  9. Criar software multithreaded.

Considerações finais:

  • O mercado de trabalho dispõe de muitas oportunidades para desenvolvedores web, programar para web "está na moda" e muitas empresas, inclusive grandes empresas estão cada vesz mais buscando profissonais com essa característica.
  • Um bom programador é aquele que além de escrever um bom e eficiente software é também aquele que é capaz de entender um código de autor desconhecido e saber julgar seus padrões de projetos, sua arquitetura.
